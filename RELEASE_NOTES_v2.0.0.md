# ARK — Traduzione italiana v2.0.0

Questa release conclude la revisione editoriale manuale dell'intero catalogo italiano di ARK: Survival Ascended.

## Cosa cambia

- 38.751/38.751 voci lette e classificate manualmente.
- 11.351 voci riscritte rispetto alla v1.2.0.
- Eliminati gli ultimi residui inglesi sospetti e corrette traduzioni letterali, innaturali o incoerenti con il contesto e la terminologia di ARK.
- Verificata la build Steam 24230788, senza nuove chiavi ufficiali rispetto alla build precedente.
- Autodetect migliorato per sistemi con più librerie o installazioni Steam.
- Backup e ripristino irrobustiti con percorsi registrati e controlli SHA-256.

## Installazione

Scarica il solo file `ARK-Survival-Ascended-Traduzione-Italiana.exe`, chiudi ARK e avvialo. L'installer rileva la copia regolarmente installata, controlla la build, crea il backup e applica il PAK italiano incorporato.

Il repository e la Release non includono file originali del gioco. È necessaria una copia regolarmente installata di ARK: Survival Ascended.

## Verifiche

- 38.751 chiavi compilate e riestratte senza differenze.
- PAK con un solo `ShooterGame.locres`, verificato byte per byte.
- Zero errori strutturali, zero avvisi, zero decisioni aperte.
- Installer autonomo one-file verificato sulla build 24230788.
